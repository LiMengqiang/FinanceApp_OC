# 期货期权行情 App 设计

目标：创建一个 Objective-C UIKit iOS App，包含“行情 / 自选 / 我的”三个 Tab。

## 数据方案

- 新浪行情源：用于第一版实时期货连续合约行情，请求 `https://hq.sinajs.cn/list=...`。
- 交易所公开数据：用于合约资料、结算价、历史日行情入口，先在 App 内提供入口和服务层封装。
- 期权：第一版先保留交易所公开数据入口和模型扩展位，避免依赖不稳定的免费实时期权接口。

## 页面

- 行情：展示常用期货连续合约，支持下拉刷新，点选添加/取消自选。
- 自选：读取本地自选合约并刷新行情。
- 我的：展示数据源说明、交易所公开数据入口、免责声明。

## 架构

- `Models` 存行情模型。
- `Services` 存新浪行情服务、自选存储、交易所公开数据入口。
- `ViewControllers` 存三个 Tab 页面和根 Tab 控制器。

## 约束

- 不使用 CocoaPods。
- 不主动提交 git。
- 不做编译校验。
